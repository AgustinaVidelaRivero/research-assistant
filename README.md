# Research Assistant

Multi-agent research system with human-in-the-loop validation, orchestrated with **LangGraph** and **Azure OpenAI** as the LLM provider.

## Status: 🚧 In development

## Stack

- Python 3.11+
- LangGraph
- Azure OpenAI
- Pydantic
- uv (dependency management and environments)

## Setup

```bash
git clone <https://github.com/AgustinaVidelaRivero/research-assistant.git>
cd research-assistant
uv sync --all-extras
cp .env.example .env
# Edit .env with your Azure OpenAI credentials
```
